# techpmi.github.io

Репозиторий для frotnend-части проекта технологической практики группы ПМИ-11МО


# Как делать изменения

1) выкачиваем свежую ветку `main`
2) создаем от нее новую ветку
3) делаем изменения в своей ветке
4) когда готово, пушим свою ветку и делаем pull-request в `main`

Перед первым запуском:

 `npm i`

Для запуска:

`npm run start` 

Собрать в продакшен:

`npm run build` 


## про деплой итога на gh-pages

В ветке `gh-pages` лежат файлы для деплоя на techpmi.github.io

Для деплоя:
- переключиться на ветку `gh-pages`
- смержить в нее ветку `main`
- выполнить `npm run build` 
- удалить из корня проекта папку `static` и файлы `asset-manifest.json`, `index.html`
- переместить содержимое папки `build` в корень
